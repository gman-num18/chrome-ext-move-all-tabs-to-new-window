# chrome-ext-move-all-tabs-to-new-window

move all tabs to new window(chrome addon)

ワンタッチで、「固定されておらず、いまアクティブなウィンドウに存在する全部のタブを、とりあえず新しいタブに移動する」だけのアドオンです。

設定項目名はありません。

## このアドオンを作った理由

調べものをしていて、Google検索をして、ECサイトを開き、さらに個別の商品ページを開いて…

気付いたら、その作業をする前に開いていたタブと、ゴチャゴチャになってしまいました。ムッキー！腹が立つ！

そこで、このアドオンです。ボタンを押すだけで、ピン止めに影響を与えず。手でがんばってタブを選択することもなしに、即座に別ウィンドウに逃がせます。
